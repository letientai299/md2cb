# Link Edge Cases

Basic link: [example](http://example.com)

Link with title: [example](http://example.com "Example Site")

Autolink: <http://example.com>

Email autolink: <user@example.com>

Reference link: [ref][1]

[1]: http://example.com "Reference"

Inline link with special chars: [test](http://example.com/path?q=a&b=c#hash)

Bare URL (should NOT autolink without <>): http://example.com

Link with **bold text**: [**bold link**](http://example.com)

Nested formatting: [*italic* and `code`](http://example.com)
